다양한 메모 - 무엇이든 메모

텍스트, 사진, 음성, 손글씨, 스크린샷, 클립보드까지 무엇이든 원래 형태 그대로 메모하세요!

퀵 메모 - 지금 바로 메모

폰 화면 위에서 아래로 손가락을 내려보세요! 알림 영역의 퀵메모로 잊기 전에 지금 바로 메모!

플로팅 메모 - 언제 어디서나 메모

이 앱 봤다, 저 앱 봤다, 메모 확인 번거롭다면? 플로팅 메모로 언제 어디서나 메모하고 보세요!

편리한 검색 - 쉽게 찾는 메모

타입별로 태그별로 비슷한 메모를 한데 모아 보고 싶은 메모를 쉽고 빠르게 찾아보세요!

부가 기능 - 다양한 알메모 부가 기능

  • 다이렉트 메모
    캡처, 클립보드, 촬영한 사진을 알메모에 바로 저장하세요
  • 추천 태그
    주소, 전화번호, 여행 등 본문 대표 태그가 자동으로 붙어 더 쉽게 찾을 수 있어요.
  • 쉬운 검색
    타입, 태그, 키워드 검색 등 다양한 방법으로 원하는 메모를 빠르게 찾으세요.
  • 중요 메모
    언제라도 바로 꺼내봐야 하는 메모는 중요 메모로 표시하여 관리하세요.
  • 메모 합치기
    비슷한 내용의 메모가 여러 개 있다면 하나로 합쳐서 더 깔끔하게 정리하세요.
  • 다한 위젯
    쓰기 아이콘, 리스트 등 원하는 위젯을 설정하여 쉽게 메모하고 기억하세요.
